◆ upo_random_string()

char * upo_random_string ( char *  s,
size_t  n 
)

Fills the given string with random characters.

Parameters
sThe string to be filled.
nThe length of the string.

The given string s is filled with n random characters taken from the printable ASCII character-encoding scheme (i.e., from ' ' to '~'). Also, an end-of-string character is added at the end of the string (i.e., at position n).
