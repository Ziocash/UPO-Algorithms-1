Detailed Description

The Stack abstract data type.

A Stack is an abstract data type that represents a container of elements such that:

  1. Each element is added over the top of the Stack (push operation).
  2. Each element is removed from the top of the Stack (pop operation).

A Stack is often called a Last-in First-out (LIFO) container since the last element that is added is the first to be removed.
